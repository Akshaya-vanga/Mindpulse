# MindPulse

MindPulse is a web-based experimental application designed to observe how users respond to scenario-based questions and simple cognitive tasks.

## Features
- IQ Test
- Mood Analysis
- Personality Observation
- Simple Interactive Games

## Technology Used
- HTML
- CSS
- JavaScript
- GitHub Pages

## Disclaimer
This project is created for educational and experimental purposes only.
The results shown are not scientifically or psychologically accurate.
